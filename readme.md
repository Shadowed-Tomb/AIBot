## `[+] INSTALLATION AND SETUP`

### `STEP 1: CLONE AND INSTALL DEPENDENCIES`

```bash
# RESTORE THE DEPENDENCIES DIRECTORY LOCALLY
npm install
```

### `STEP 2: ENVIRONMENT CONFIGURATION (.ENV)`

`CREATE A .ENV FILE IN THE ROOT DIRECTORY AND CONFIGURE THE KEYS:`

```env
TELEGRAM_BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN
PUTER_AUTH_TOKEN=YOUR_PUTER_AUTH_TOKEN
FAL_KEY=YOUR_FAL_API_KEY
OWNER_ID=YOUR_TELEGRAM_USER_ID
TELEGRAM_API_ID=YOUR_TELEGRAM_API_ID
TELEGRAM_API_HASH=YOUR_TELEGRAM_API_HASH
```

## `[+] BOT COMMAND INTERFACE`

### `GENERAL USER COMMANDS`
- `/START` : `BOOT THE BOT AND DISPLAY CONTEXT-AWARE COMMANDS LIST.`
- `/HELP` : `SHOW DETAILED CAPABILITIES MENU.`
- `/IMAGINE <PROMPT>` : `GENERATE IMAGE USING GROK IMAGINE.`
- `/ANIMATE <PROMPT>` : `GENERATE VIDEO FROM PROMPT OR BY REPLYING TO A PHOTO.`
- `/RESET` : `CLEAR PERSONAL CHAT MEMORY HISTORY.`

### `ADMIN COMMANDS`
- `/BAN <USERID_OR_USERNAME>` : `BLOCK A USER FROM USING THE BOT AND KICK FROM GROUP.`
- `/UNBAN <USERID_OR_USERNAME>` : `UNBAN A PREVIOUSLY BANNED USER.`
- `/BANLIST` : `LIST ALL BANNED USERS.`
- `/ADMINLIST` : `LIST OWNER AND ACTIVE BOT ADMINS.`

### `OWNER-ONLY COMMANDS`
- `/ADDADMIN <USERID_OR_USERNAME>` : `PROMOTE A USER TO BOT ADMIN.`
- `/DELADMIN <USERID_OR_USERNAME>` : `DEMOTE AN ACTIVE BOT ADMIN.`
